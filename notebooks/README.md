# End-to-end inference notebooks

The double-anonymous review package is intentionally lightweight and focuses on deterministic reproduction of the submitted numerical analyses from the fixed per-image scored outputs in `../results/`.

The consolidated end-to-end inference notebooks require environment-specific public-dataset mounts and SAM/MedSAM checkpoint paths. They will be included, together with complete setup instructions, in the archival public repository released upon acceptance.

The analysis scripts in `../analysis_scripts/` are sufficient to reproduce the principal manuscript tables, confidence intervals, cross-modality bootstrap results, and the reported sensitivity/robustness summaries included in this review package.
